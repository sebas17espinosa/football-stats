# Football Stats ⚽

Proyecto personal de análisis de estadísticas de fútbol usando SQL Server y Power BI.
Repositorio creado como parte de mi aprendizaje del stack: Git + SQL Server + Power BI.

## Contenido del repositorio

- `data/` — Archivos CSV con datos crudos de fútbol
- `sql/` — Scripts de creación de tablas y consultas
- `powerbi/` — Reportes y dashboards

## Datasets incluidos

| Archivo | Descripción |
|---|---|
| `top_goleadores_historicos.csv` | Top 20 goleadores históricos de selecciones nacionales |

## Stack utilizado

- **Git + GitHub** — Control de versiones
- **SQL Server** — Base de datos y consultas
- **Power BI** — Visualización y dashboards
